
# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Build the classic Hangman game using Python. In this assignment, you will practice working with strings, loops, conditionals, user input, and random word selection.

## 📝 Tasks

### 🛠️	Build the Core Game Loop

#### Description
Create a Hangman game that chooses a random word from a predefined list and lets the player guess one letter at a time. The game should continue until the player guesses the full word or runs out of incorrect attempts.

#### Requirements
Completed program should:

- Randomly select a word from a predefined list.
- Accept letter guesses from the player one at a time.
- Display the current progress using underscores for unguessed letters.
- Track and update the number of incorrect guesses remaining.
- End the game when the word is fully guessed or the player uses all attempts.


### 🛠️	Add Game Feedback and Rules

#### Description
Improve the player experience by adding clear feedback after each guess. Make sure the game explains what happened, shows remaining attempts, and displays a final message for either a win or a loss.

#### Requirements
Completed program should:

- Show whether each guessed letter is correct or incorrect.
- Prevent or handle repeated guesses in a clear way.
- Display the current word progress after every turn.
- Show a win message when the player guesses the full word.
- Show a lose message that reveals the correct word when attempts run out.
